# Employee Leave Management System

## Features
- Employee Registration/Login
- Apply Leave
- Leave History
- Admin Dashboard
- Approve/Reject Leave
- Position-wise Leave Tracking
- Email Notifications

## Tech Stack
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- HTML
- CSS
- JavaScript

- leave_requests
